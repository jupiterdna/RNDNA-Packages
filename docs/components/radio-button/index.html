Radio Button

This component meant for easy selection and decision-making. By convention, only one button at a time can be selected

Usage​

Import

import { DNA } from '@rndna/radiobutton';

Props​

NameTypeDefaultDescription
idstringundefinedProperty that is used to pass in the id parameter of code.
valuestringundefinedProperty that defines the value of the selected option.
labelstringundefinedProperty that defines the label of the option.
disabledbooleanfalseProperty where the user can enable or disable the certain option
checkedbooleanfalseProperty where the user can checked or unchecked the certain option
sizexs, sm, md, lg, xlmdProperty that defines the size of the component or a selection.
onPress((event: GestureResponderEvent) => void), undefinedundefinedHandler when the component is pressed
